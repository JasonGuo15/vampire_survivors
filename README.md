# Vampire Survivor

## Game Description
Vampire Survivor is an exciting action-adventure game where players fight against waves of terrifying vampire foes, utilizing various weapons and strategies to survive. The game features stunning graphics, engaging gameplay, and a unique storyline that keeps players on the edge of their seats.

## Installation Instructions
1. Clone the repository:  
   ```bash
   git clone https://github.com/JasonGuo15/vampire_survivor.git
   ```
2. Navigate to the project directory:  
   ```bash
   cd vampire_survivor
   ```
3. Install the necessary dependencies:  
   ```bash
   npm install
   ```
4. Run the game:  
   ```bash
   npm start
   ```

## Controls
- **Movement**: Arrow keys or WASD
- **Attack**: Left mouse button
- **Special Ability**: Right mouse button
- **Pause**: Esc

## Game Mechanics Overview
In Vampire Survivor, players must:
- Defend against continuously spawning vampire enemies.
- Collect power-ups that enhance abilities and weapons.
- Manage resources like health and ammunition to survive.
- Navigate through various levels with increasing difficulty.
